DomainWHOIS
----
Simple Domain Information Gathering Tool

Results
----
- Name
- IP
- TLD
- Country
- City
- Region
- Latitude
- Longitude
- Registrar
- Registrant country
- Creation date
- Expiration date
- Last updated
- Status
- Statuses
- DNSSEC
- Name servers
- ASN
- ZIP/Postal code
- Registrant
- Emails

Tested on
----
Termux

Requirements
----
Python

`pip install whois`

`pip install geocoder`

Installation
----
`apt install git`

`git clone https://github.com/t0mxplo1t/DomainWHOIS.git`

`cd DomainWHOIS`

`python get.py`

Example
----
`google.com`

`facebook.com`

`twitter.com` etc...
